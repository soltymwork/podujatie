Prečo nesmieš chýbať?

Či ste skúsený bežec, hobby nadšenec alebo hľadáte program pre celú rodinu, toto podujatie je pre vás!

Pripravili sme kvalitné bežecké trate, bohatý sprievodný program a zábavu pre všetkých. Naším cieľom je spojiť šport, komunitu a dobrú náladu do nezabudnuteľného dňa.

  • Účastnícka medaila pre každého finišera
  • Merané preteky s čipovým systémom
  • Tombola s hodnotnými cenami
  • Zábava a atrakcie pre deti po celý deň
  • Večerný hudobný program
